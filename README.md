# Personal Web Profile

A simple personal web profile built with HTML, CSS, and JavaScript.

## Overview

This is a single-page personal website that includes:

- Header with navigation menu and music toggle button
- About Me section with an interactive fun fact button (with confetti effect!)
- Hobbies & Interests section with hover effects
- Contact Info section with social media links
- Responsive design for mobile and desktop
- Custom background patterns and animated banner
- Google Fonts (Silkscreen) typography

## Technologies Used

- **HTML5** - Semantic HTML structure
- **CSS3** - Styling, animations, responsive design, and background images
- **JavaScript** - Interactive elements, confetti effect, dancing text, and audio control

## Features

- Navigation menu with smooth scrolling to sections
- Interactive fun fact button that displays random facts with confetti explosion
- Dancing banner text animation on page load and button click
- Background lofi music player with play/pause toggle
- Hover effects on buttons and hobby cards
- Repeating background pattern (`assets/background.png`)
- Banner image (`assets/banner.gif`) that scales responsively
- Custom favicon (`assets/favicon.png`)
- Google Fonts integration (Silkscreen - pixelated font)
- Mobile-friendly responsive layout
- Clean, modern design with animations

## How to Use

1. Click the 🎵 button in the navigation to play/pause background lofi music
2. Click "Click for a Fun Fact!" to see confetti and dancing text animation
3. Navigate between sections using the menu links
